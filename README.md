# anri_couturi
Das hier ist eine Demo-Website, die ich selbst gebaut habe. Bei ein paar JavaScript-Funktionen und kleinen CSS/HTML-Elementen habe ich KI-Tools als Hilfe genommen, aber den Code danach noch selbst angepasst. Die Bilder von GUCCI kommen direkt aus dem offiziellen Webstore und sind nur für Demo- und Lernzwecke da – nicht für irgendwas Kommerzielles. Alle Logos und Schriftzüge der fiktiven Marke „anri couturi“ habe ich eigenhändig in Figma erstellt, und die Icons findest du auf https://iconmonstr.com.

Randnotiz: Über externe Links eingebundene Schriftarten (z. B. Google Fonts) können auf GitHub Pages teilweise anders geladen werden als lokal auf meinem Rechner. Die primären Funktionen der Seite bleiben jedoch vollständig erhalten. Die Fonts sind eher nur kosmetisch.
